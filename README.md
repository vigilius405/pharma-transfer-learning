# pharma-transfer-learning
Final project for Distributed Optimization.
Information on datasets here: https://moleculenet.org/datasets-1
Information on UME here: https://huggingface.co/karina-zadorozhny/ume-base

Running pharma_ftl.ipynb will enable training of one party, all-reduce, FedBCD, and FedCVT models (toy classifiers are small), and will produce figures used in the project. Environment requirements are in requirements.txt.
